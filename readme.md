# win-screensaver

> Start the screensaver in Windows systems.


## Install

```
$ npm install --save win-screensaver
```


## Usage

```js
var winScreensaver = require('win-screensaver');

winScreensaver(function (err) {
	console.log('Screensaver started');
});
```
